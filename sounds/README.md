# Sounds

Drop these 6 looping MP3 files into this folder. Each should be **30 seconds – 5 minutes** long. The player loops them seamlessly.

| File | Channel | Source suggestion |
|---|---|---|
| `rain.mp3` | Rain (always-on) | https://pixabay.com/sound-effects/search/rain/ |
| `thunder.mp3` | Thunder | https://pixabay.com/sound-effects/search/thunder/ |
| `wind.mp3` | Wind | https://pixabay.com/sound-effects/search/wind/ |
| `waves.mp3` | Ocean waves | https://pixabay.com/sound-effects/search/ocean/ |
| `fire.mp3` | Fireplace crackle | https://pixabay.com/sound-effects/search/fireplace/ |
| `music.mp3` | Ambient music | https://pixabay.com/music/search/ambient/ |

**Tip:** trim each file to start and end on near-silence so the loop point is inaudible. Free editor: [Audacity](https://www.audacityteam.org/).

Pixabay is royalty-free with no attribution required. All other sources you use must be cleared for commercial reuse.
